# Tooltips

Leafy UI tooltips provide contextual information on hover or focus, perfect for explaining UI elements without cluttering the interface.

## Class Reference

| Class Name | Type | Description |
|------------|------|-------------|
| `tooltip` | Component | Base class for the tooltip component |
| `tooltip-content` | Component | Rich content container element |
| `tooltip-open` | State | Force tooltip to always be visible |
| `tooltip-top` | Position | Position tooltip above element |
| `tooltip-bottom` | Position | Position tooltip below element |
| `tooltip-left` | Position | Position tooltip to the left |
| `tooltip-right` | Position | Position tooltip to the right |
| `tooltip-bg-{color}` | Color | Background color for tooltip |
| `tooltip-text-{color}` | Color | Text color for tooltip |
