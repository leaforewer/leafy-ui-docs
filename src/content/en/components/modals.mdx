---
title: Modals
desc: Accessible dialog boxes built on native HTML dialog elements with smooth animations and backdrop effects.
classnames:
  component:
    - class: modal
      desc: Base modal container with backdrop and positioning
    - class: modal-box
      desc: Content container with styling and animations
    - class: modal-backdrop
      desc: Clickable backdrop area for closing modal
  modifiers:
    - class: modal-open
      desc: Force modal to show (for testing/demos)
  attributes:
    - class: showModal()
      desc: JavaScript method to open a dialog modal
    - class: close()
      desc: JavaScript method to close a dialog modal
    - class: method="dialog"
      desc: Form method for closing modal when submitted
---

import { Tabs } from 'nextra/components'

# Modals

Accessible dialog boxes built on native HTML `<dialog>` elements with smooth animations and backdrop effects.

## Setup

Choose your setup method based on your framework:

<Tabs items={['JavaScript']}>
  <Tabs.Tab>
    First, install the Leafy UI core package:
    
    ```bash
    npm install @leafy-ui/core
    ```
    
    Then import the modal styles in your project:
    
    ```js
    import '@leafy-ui/core/css/components/modal.css'
    ```
  </Tabs.Tab>
</Tabs>
