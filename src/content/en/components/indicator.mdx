# Indicator

Position indicators and badges relative to other elements for notifications and status displays. Perfect for notification counters, status dots, and contextual information.

## Class Reference

| Class Name | Type | Description |
|------------|------|-------------|
| `indicator` | Component | Container that positions indicator items relative to its content |
| `indicator-item` | Component | The positioned indicator (badge, dot, etc.) |
| `badge` | Utility | Styled badge component for notifications |
| `badge-bg-{color}` | Utility | Background color variants for badges |
| `badge-text-{color}` | Utility | Text color variants for badges |

## Usage Notes

- The `indicator` class creates a relative positioning container
- `indicator-item` is positioned absolutely in the top-right corner by default
- Combine with `badge` classes for styled notification badges
- Use color variants like `badge-bg-red-500` and `badge-text-white` for theming
- Perfect for notification counters, status indicators, and contextual badges
- Works with any content: icons, buttons, cards, avatars, etc.
