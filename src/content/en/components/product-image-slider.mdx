# Product Image Slider

Interactive product image slider with thumbnails, zoom functionality, and customizable track indicators. Perfect for e-commerce product galleries and detailed image viewing.

> **SSR-safe:** This component reads images from the `data-images` attribute or from direct child `<img>` elements. No client mutation is required, so it avoids hydration mismatches.

## Class Reference

| Class Name | Type | Description |
| --- | --- | --- |
| `product-image-slider` | Component | Main product image slider web component container |

### Attributes

| Attribute | Type | Description |
| --- | --- | --- |
| `strip` | Boolean | Display thumbnail strip below main image |
| `zoom` | Boolean | Enable tap-to-zoom fullscreen (single tap/click on image area) |
| `tracks` | Boolean | Show track indicators for navigation |
| `zoomBtn` | Boolean | Display zoom button overlay (click to fullscreen) |
| `trackstyle` | String | Track indicator style: `"pill"`, `"line"`, `"skewed"`, `"dot"` |

### Utility Classes

| Class Name | Description |
| --- | --- |
| `carousel-track-bg-[color]` | Set track indicator background color |
| `carousel-active-track-bg-[color]` | Set active track indicator background color |
| `carousel-thumb-active-border-[color]` | Set active thumbnail border color |

## JavaScript API

You can still set images **programmatically** (prop is bound to `data-images` under the hood):

```js
const slider = document.querySelector('product-image-slider');
slider.images = [
  { url: 'https://images.unsplash.com/photo-1542291026-7eec264c27ff?w=1200&q=60', alt: 'Front view' },
  { url: 'https://images.unsplash.com/photo-1512436991641-6745cdb1723f?w=1200&q=60', alt: 'Side view' },
  { url: 'https://images.unsplash.com/photo-1519741497674-611481863552?w=1200&q=60', alt: 'Detail' }
];
```
