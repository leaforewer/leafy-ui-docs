# Type Writer

Animated typewriter effect that types text character by character. Perfect for hero sections, loading states, and engaging text reveals.

## Class Reference

| Class Name | Type | Description |
|------------|------|-------------|
| `type-writer` | Component | Main typewriter web component |

### Attributes

| Attribute | Type | Description |
|-----------|------|-------------|
| `text` | String | Text content to be typed |
| `delay` | Number | Delay before animation starts (milliseconds) |
| `speed` | Number | Typing speed in milliseconds per character |
| `inview` | Boolean | Start animation when element enters viewport |
| `blink` | Boolean | Enable blinking cursor effect |
| `caretChar` | String | Custom cursor character (default: "|") |
| `loop` | Boolean | Loop the animation continuously |

## Javascript API

### Options

```html

<type-writer id="tw"></type-writer>

<script type="module">
  const tw = document.getElementById('tw');

  // TEXT (string or array for multi-line)
  tw.text = ['Hello there,', 'this types like a terminal.'];

  // OPTIONS (ms per char — larger is slower)
  tw.speed = 40;          // base speed
  tw.variance = 0.35;     // natural jitter (0..1)
  tw.commaPause = 120;    // pause after ','
  tw.periodPause = 240;   // pause after . : ; ! ?
  tw.blink = true;        // caret blink
  tw.caretChar = '';      // e.g. '_' to use a char caret
  tw.inview = true;       // start when visible
  tw.delay = 300;         // start delay (ms) once starting
  tw.loop = false;        // repeat animation

  // CONTROLS
  tw.play();            // start immediately (ignores inview)
  tw.pause();           // pause
  tw.reset();           // clear
  tw.restart();         // reset + play

  // EVENTS
  tw.addEventListener('tw-start',  () => console.log('started'));
  tw.addEventListener('tw-finish', () => console.log('finished'));
</script>

```
