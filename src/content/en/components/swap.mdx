# Swap

Interactive toggle switches with smooth animations for state changes. Perfect for theme toggles, view switches, and any binary state controls.

## Class Reference

| Class Name | Type | Description |
|------------|------|-------------|
| `swap` | Component | Base class for creating toggle switches |
| `swap-on` | Component | Content shown when checkbox is checked |
| `swap-off` | Component | Content shown when checkbox is unchecked |
| `swap-rotate` | Modifier | Adds rotation animation between states |
| `swap-flip` | Modifier | Adds flip animation between states |

## Usage Notes

- Use an `<input type="checkbox">` inside the `swap` label for keyboard accessibility
- Content inside `swap-on` is shown when the checkbox is checked
- Content inside `swap-off` is shown when the checkbox is unchecked
- Combine with animation modifiers like `swap-rotate` or `swap-flip` for smooth transitions
- Perfect for theme toggles, view switchers, and binary state controls
