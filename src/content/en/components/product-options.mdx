# Product Options

Product Options provides interactive selection components for colors, variants, sizes, and other product attributes. The system supports multiple option types including color swatches, image variants, and text-based selections with full accessibility support.

## Class Reference

| Class Name | Type | Description |
|------------|------|-------------|
| `product-options` | Component | Base container for product option groups with proper spacing and layout |
| `options-color` | Modifier | Enables color-specific option styling with CSS custom properties |
| `option` | Component | Individual option container with proper radio input styling |
| `option-active-border-*` | Modifier | Sets active option border color |
| `option[data-checked]` | State | Checked state styling for option containers |
| `option input[type="radio"]` | Utility | Hidden radio input with accessibility features |
| `option label` | Utility | Clickable label area with hover and focus states |

## Accessibility Features

- **ARIA Support**: Uses `role="radiogroup"` and `aria-label` for screen readers
- **Keyboard Navigation**: Full keyboard support for radio button groups
- **Focus Management**: Proper focus indicators and tab navigation
- **Semantic HTML**: Uses proper `input[type="radio"]` and `label` elements

### CSS Custom Properties

- **`--option-color`** - Sets the color value for color swatch options (used with `.options-color`)
