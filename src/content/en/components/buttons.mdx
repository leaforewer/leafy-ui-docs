# Button

Button provides a consistent and interactive button system with multiple variants and states. The core `.pop-btn` class creates modern, accessible buttons with smooth animations and hover effects.

## Setup

Install Leafy UI and include the CSS in your project:

```bash
npm install @leafy-ui/core
```

```css
@import "@leafy-ui/core/style";
```
