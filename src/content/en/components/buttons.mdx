# Button

Standard button component with multiple sizes and color variants. The `.btn` class provides consistent styling and behavior across your application with semantic color options.

## Class Reference

| Class Name | Type | Description |
|------------|------|-------------|
| `btn` | Component | Base button class with standard styling |

### Size Modifiers

| Class Name | Description |
|------------|-------------|
| `btn-xs` | Extra small button size |
| `btn-sm` | Small button size |
| `btn-lg` | Large button size |
| `btn-xl` | Extra large button size |

### Color Variants

| Class Name | Description |
|------------|-------------|
| `btn-success` | Success state button (green) |
| `btn-info` | Info state button (blue) |
| `btn-warning` | Warning state button (yellow/orange) |
| `btn-danger` | Danger state button (red) |
