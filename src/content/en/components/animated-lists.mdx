# Animated Lists

Animated Lists provides smooth staggered animations for list items. Each item animates in sequence with customizable delays, creating engaging reveal effects for dynamic content.

## Class Reference

| Class Name | Type | Description |
|------------|------|-------------|
| `leaf-stagger` | Component | Base class for creating staggered animations on list items |
| `leaf-stagger > li` | Utility | Individual list items with animation delay based on --i variable |

## CSS Custom Properties

- **`--i`** - Index variable that controls the animation delay for each list item

## Usage Notes

- Each `<li>` element requires a `--i` CSS custom property with its index value.
- Works with any list content: text, cards, navigation items, etc.
- Animations trigger on page load or you can control it with your JS.
