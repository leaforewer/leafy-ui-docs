---
title: Dropdowns
desc: Popover-based dropdown menus with flexible positioning options built on the modern Popover API.
classnames:
  component:
    - class: dropdown
      desc: Base dropdown container with positioning and animations
  modifiers:
    - class: dropdown-top
      desc: Positions dropdown above the trigger element
    - class: dropdown-bottom
      desc: Positions dropdown below the trigger element (default)
    - class: dropdown-center
      desc: Centers dropdown horizontally relative to trigger
    - class: dropdown-open
      desc: Force dropdown to show (for testing/demos)
  attributes:
    - class: popover
      desc: Native popover attribute for dropdown behavior
    - class: popovertarget
      desc: Connects button to specific popover element by ID
---

import { Tabs } from 'nextra/components'

# Dropdowns

Popover-based dropdown menus with flexible positioning options. Built on the modern Popover API for better accessibility and performance.

## Setup

Choose your setup method based on your framework:

<Tabs items={['JavaScript']}>
  <Tabs.Tab>
    First, install the Leafy UI core package:
    
    ```bash
    npm install @leafy-ui/core
    ```
    
    Then import the dropdown styles in your project:
    
    ```js
    import '@leafy-ui/core/css/components/dropdown.css'
    ```
  </Tabs.Tab>
</Tabs>
