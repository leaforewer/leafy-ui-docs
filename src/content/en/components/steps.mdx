# Steps

Leafy UI steps provide visual progress indicators perfect for multi-step forms, wizards, and process flows.

## Class Reference

| Class Name | Type | Description |
|------------|------|-------------|
| `steps` | Component | Base class for the steps component |
| `step` | Component | Individual step item |
| `completed` | State | Marks step as completed/active |
| `steps-vertical` | Layout | Vertical orientation modifier |
| `steps-hover` | Behavior | Enable hover effects on incomplete steps |
| `steps-bg-{color}` | Color | Background color for inactive steps |
| `steps-text-{color}` | Color | Text color for inactive steps |
| `steps-active-bg-{color}` | Color | Background color for active/completed steps |
| `steps-active-text-{color}` | Color | Text color for active/completed steps |
