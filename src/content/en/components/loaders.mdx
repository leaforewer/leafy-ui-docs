# Loading

Various loading spinner animations for different states and contexts. Provides multiple animation styles with customizable sizes and colors.

## Class Reference

| Class Name | Type | Description |
|------------|------|-------------|
| `loading` | Component | Base loading spinner animation |
| `loading-bars` | Modifier | Bar-style loading animation |
| `loading-dots` | Modifier | Dot-style loading animation |
| `loading-infinity` | Modifier | Infinity symbol loading animation |
| `loading-pulse` | Modifier | Pulsing circle loading animation |
| `loading-lg` | Size | Large size variant |
| `loading-xl` | Size | Extra large size variant |

## Usage Notes

- Use `loading` as the base class for all spinner animations
- Combine with style modifiers: `loading-bars`, `loading-dots`, `loading-infinity`, `loading-pulse`
- Size variants: `loading-lg` (large), `loading-xl` (extra large)
- Customize colors using Tailwind text color classes like `text-orange-500`
- Perfect for loading states, async operations, and progress indicators
- All animations are CSS-based for optimal performance
