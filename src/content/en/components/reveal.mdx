# Reveal

Scroll-based reveal animations that trigger when elements enter the viewport. Uses modern CSS animation-timeline for smooth, performant animations.

## Class Reference

| Class Name | Description |
|------------|-------------|
| `reveal-up` | Fade up animation on scroll into view |
| `reveal-left` | Slide in from left animation on scroll |
| `reveal-fade` | Fade in animation on scroll into view |

### Custom Properties

| Property | Description |
|----------|-------------|
| `--reveal-delay` | Custom delay for reveal animation (default: 100ms) |

## Browser Support

Reveal animations use modern CSS `animation-timeline: view()` which provides:

- **Progressive Enhancement**: Animations only work in supported browsers
- **Graceful Fallback**: Content remains visible and accessible in all browsers
- **Performance**: Hardware-accelerated animations tied to scroll progress

### Supported Browsers

- Chrome 115+
- Edge 115+
- Safari 16.4+
- Firefox (experimental support)

## Best Practices

```html
<!-- Use consistent spacing for better visual flow -->
<div class="space-y-8">
  <div class="reveal-up">First element</div>
  <div class="reveal-up">Second element</div>
</div>

<!-- Stagger delays for sequential animations -->
<div class="reveal-up" style="--reveal-delay: 0ms">Element 1</div>
<div class="reveal-up" style="--reveal-delay: 100ms">Element 2</div>
<div class="reveal-up" style="--reveal-delay: 200ms">Element 3</div>

<!-- Combine with other utility classes -->
<div class="reveal-fade bg-gradient-to-r from-blue-500 to-purple-600 text-white p-8 rounded-xl">
  Styled reveal element
</div>
```

## Framework Integration

### React

```jsx
function RevealExample() {
  return (
    <div className="space-y-6">
      <div className="reveal-up bg-blue-100 p-6 rounded-lg">
        This fades up on scroll
      </div>
      <div 
        className="reveal-fade bg-green-100 p-6 rounded-lg"
        style={{ '--reveal-delay': '200ms' }}
      >
        This fades in with delay
      </div>
    </div>
  );
}
```
