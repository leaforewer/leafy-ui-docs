# Accordion

Accordion provides collapsible content sections with customizable icons and smooth animations. Built on native HTML `<details>` elements for accessibility.

## Setup

Install Leafy UI and include the CSS in your project:

```bash
npm install @leafy-ui/core
```

```css
@import "@leafy-ui/core/style";
```

### Basic accordion

<Tabs items={['Preview', 'HTML']}>
  <Tabs.Tab>
    <div className="not-prose my-6">
      <div dangerouslySetInnerHTML={{
        __html: `
          <details class="accordion accordion-icon-chevron" name="demo-group-1">
            <summary>Accordion Item 1 (Chevron Icon)</summary>
            <p>This accordion uses a chevron icon that rotates when opened.</p>
          </details>
          <details class="accordion accordion-icon-chevron" name="demo-group-1">
            <summary>Accordion Item 2 (Chevron Icon)</summary>
            <p>This is the content of the second accordion item.</p>
          </details>
        `
      }} />
    </div>
  </Tabs.Tab>
  <Tabs.Tab>
    ```html
    <details class="accordion accordion-icon-chevron" name="accordion-group">
      <summary>Accordion Title</summary>
      <p>Accordion content goes here.</p>
    </details>
    ```
  </Tabs.Tab>
</Tabs>
