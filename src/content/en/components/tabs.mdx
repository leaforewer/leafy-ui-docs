# Tabs

Interactive tab navigation with customizable styles and smooth transitions. Built with radio inputs for accessibility and keyboard navigation support.

## Class Reference

| Class Name | Type | Description |
|------------|------|-------------|
| `tabs` | Component | Main tabs container |
| `tab` | Component | Individual tab button (applied to radio input) |
| `tab-content` | Component | Content panel for each tab |

### Modifiers

| Class Name | Description |
|------------|-------------|
| `tabs-border` | Add border styling to tabs |
| `tabs-lift` | Elevated tab style with shadow effect |

### Utility Classes

| Class Name | Description |
|------------|-------------|
| `tabs-lift-border-[color]` | Set custom border color for lifted tabs |
| `tabs-bg-[color]` | Set tab background color |
| `tabs-fg-[color]` | Set tab text color |
| `tabs-fg-active-[color]` | Set active tab text color |
| `tabs-border-[color]` | Set border color for bordered tabs |

## Accessibility Features

The tabs component is built with accessibility in mind:

- **Keyboard Navigation**: Use arrow keys to navigate between tabs
- **ARIA Labels**: Each tab should have a descriptive `aria-label`
- **Radio Input Pattern**: Uses radio inputs for semantic tab behavior
- **Focus Management**: Automatic focus management when switching tabs
